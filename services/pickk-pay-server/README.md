## Description

[PICKK](https://pickk.one) Payments Server (Nest.js + MySQL)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# start mysql
$ npm run set

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run jest

# e2e tests
$ npm run jest:e2e

# test coverage
$ npm run jest:cov

# format, lint, build, jest
$ npm run test
```

## Features

- authentication : `/auth`
- payments: `/payments`
